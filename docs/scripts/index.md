# Scripts & Tools

This section will document useful browser extensions, userscripts and community tools for MissionChief UK.

## Inclusion standard

Every tool guide must identify:

- official source and maintainer;
- installation method;
- permissions and data access;
- supported browsers and devices;
- current maintenance status;
- compatibility with other major tools;
- recommended UK configuration;
- safe disable, recovery and removal steps.

## Priority guides

- Tampermonkey installation and userscript safety
- LSSM V.4 and recommended UK modules
- MissionChief Map Command Toolkit
- MissionChief Command Nexus
- Common script conflicts and duplicate UI detection
- Desktop, Android and iOS/Safari compatibility

!!! warning "Security"
    Never install a userscript from an unverified copy, shortened link or pasted code whose origin cannot be established. Userscripts can read and alter pages opened in their allowed scope.
