# Ambulance Service

## Operational overview

Ambulance operations combine patient demand, treatment capacity, transport destinations and specialist clinical resources. Expansion should balance unit volume with hospital access and realistic travel times.

!!! info "Evidence status"
    This page defines the final guide structure. Numerical claims must be verified before being promoted to factual reference material.

## Command priorities

1. Build sufficient routine ambulance capacity before adding specialist units.
2. Position resources around mission density and hospital access.
3. Monitor transport bottlenecks as account demand rises.
4. Match specialist vehicles to trained personnel.
5. Treat patient throughput as a network problem rather than a station-only problem.
