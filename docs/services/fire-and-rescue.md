# Fire and Rescue Service

## Operational overview

The Fire and Rescue Service is the core expansion route for most MissionChief UK accounts. This guide will cover station progression, appliance composition, specialist extensions, personnel qualifications, mission scaling and fleet planning.

!!! info "Evidence status"
    This page is an editorial framework. Exact costs, unlock thresholds and mission requirements must be verified against the current UK game before publication as factual data.

## Command priorities

1. Maintain enough frontline pumping appliances to absorb routine demand.
2. Add specialist capability only when staffing and dispatch coverage are sustainable.
3. Avoid concentrating every specialist resource at one station.
4. Review transport time, not only straight-line coverage.
5. Keep trained personnel aligned with the vehicles that require them.
